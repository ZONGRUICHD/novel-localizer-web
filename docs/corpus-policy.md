# 语料、版权与外部处理规则

每个资料库必须记录：处理权声明、目标地区、`paired` 或 `style_only`、优先级、是否允许向外部模型发送短片段、确认时间与备注。未确认权限的资料库只能保存元数据，不能构建索引或进入翻译 Prompt。

当前本地“败犬女主太多了”文件是中文译本，可作为高优先 `style_only` 资料库；补充对应的无 DRM 日文原版并人工确认对齐后，才可升级为 `paired`。根目录与 `txt/` 重复副本、单卷与合集按内容哈希去重，不重复增加权重。

`lightnovel-2025` 只提供批量选择，不默认全量加入检索。败犬资料库默认优先级 100；明确选择的通用资料默认 20。

任何原书、译本、对齐中间件、Prompt 日志和导出成品都不进入 Git。仓库测试只使用无版权负担的合成句子、图片和文档。
